해당 학생의 수행평가 결과물 등급입니다. 등급에 따라 요약 표현 방식이 달라집니다.

📋 채점 기준 (루브릭)
📁 학생 수행평가 결과물 업로드
📂

클릭하거나 파일을 드래그하여 업로드

PDF · DOCX · PPTX · JPG · PNG  |  여러 파일 동시 업로드 가능

💡 사용 방법
  1. AI 제공자(Claude·ChatGPT·Gemini)를 선택하고 API 키와 과목명을 입력하세요
  2. 채점 기준(루브릭)을 입력하거나 파일로 불러오세요
    없어도 되지만, 있을수록 정확도 향상
  3. 학생 수행평가 파일(PDF·DOCX·PPTX)을 업로드하세요
    파일명 = 학생 이름으로 사용 (예: 홍길동.pdf)
  4. 수업 관찰 내용 생성 버튼 클릭
  5. 결과 카드의 📋 복사 클릭 → 생기부 작성 도우미 수업 관찰 내용란에 붙여넣기 → ✨ 생성하기
📌 출력 형식
생기부 작성 도우미가 요구하는 "~함." 종결형 관찰 문장을 줄바꿈으로 나열합니다.
각 문장에 단원명 + 구체적 활동 + 학생의 주장·분석·결론을 압축하여
도우미가 고품질 생기부 문장을 생성할 수 있도록 최적화합니다.
